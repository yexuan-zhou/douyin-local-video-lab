# 软件基本信息

## 软件名称

Local Video Lab 本地视频素材归档与AI复盘系统

## 版本号

V1.0

## 软件类型

本地视频素材管理、关键帧抽取、内容摘要与复盘报告生成系统。

## 运行环境

- 操作系统：Windows 10/11、macOS、Linux
- 运行环境：Node.js 20 或以上版本
- 浏览器环境：Microsoft Edge / Chromium，用于本地视频预览与关键帧抽取
- 可选部署环境：Docker，用于后续私有化部署和本地服务预览

## 开发语言与技术

- JavaScript / Node.js
- HTML5 / CSS3
- Chrome Extensions Manifest V3
- Playwright
- Docker
- Markdown 文档体系

## 主要功能

1. 本地视频素材归档：支持对用户本人拥有权利或已获授权的视频素材进行本地化整理。
2. 本地视频导入：支持用户将本地 MP4 等视频文件放入指定目录进行后续处理。
3. 关键帧抽取：基于本地视频文件，按固定数量或指定时间点抽取关键帧。
4. 关键帧总览：根据抽取结果生成本地 HTML 总览页面，便于查看视频画面和字幕区域。
5. 摘要生成流程：提供内容摘要、重点片段记录、标签整理和复盘结论的结构化流程。
6. 报告导出基础：支持生成 HTML、Markdown、CSV 等报告输出的基础结构。
7. Web Demo 预览：提供产品首页和样例复盘报告页面，用于展示系统功能和使用场景。
8. 私有化部署基础：提供 Docker 基础结构，为后续局域网或团队内部部署预留能力。

## 技术特点

- 本地优先：默认在用户本机处理视频素材，不主动上传用户视频。
- 合规边界清晰：仅用于处理用户本人拥有权利或已获授权的视频素材。
- 模块化结构：脚本、文档、Web Demo、扩展和 Docker 基础结构相互独立，便于维护。
- 可扩展报告体系：通过样例数据、HTML 页面和 Markdown 模板支持后续报告模板扩展。
- 轻量运行：核心流程基于 Node.js 脚本和静态页面，便于本地运行和演示。
- 可私有部署：预留 Docker 部署基础，便于未来扩展为本地 Web 服务或团队内部工具。
