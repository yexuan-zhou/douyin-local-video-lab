# Roadmap

## Phase 1: Showcase MVP

Goals:

- GitHub product page
- Web Demo page
- Demo report
- Local CLI workflow
- Compliance document
- Monetization notes

## Phase 2: Local Product

Goals:

- Windows one-click start script
- Mac one-click start script
- Local Web UI
- Batch import for local videos
- Report template selection
- HTML / CSV / Markdown export

## Phase 3: Private Deployment

Goals:

- Docker deployment
- LAN access
- Team media library
- Multi-user permissions
- Private deployment service

## Phase 4: Advanced Product

Goals:

- Desktop App
- Browser helper extension, without bypassing platform restrictions
- API
- SaaS, only after copyright, privacy, storage, and compliance issues are solved
