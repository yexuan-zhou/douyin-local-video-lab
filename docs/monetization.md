# Monetization Plan

## Positioning

中文：本地短视频素材归档、抽帧、摘要和内容复盘工具。

English: Local-first short-video archive and AI review tool for creator-owned or authorized content.

## Phase 1 Revenue Options

- Local tool setup service: ¥199 per setup
- Personal plan: ¥99 per year
- Pro plan: ¥199-299 per year
- Studio deployment: from ¥999

## Sellable Service Packages

- Media organization service
- Content review report generation
- Local tool installation
- Private team deployment
- Custom report templates
- Portfolio or graduation-project video material organization

## Directions To Avoid For Now

- Public video download SaaS
- Bulk collection services
- Platform data scraping services
- Extensions that bypass platform restrictions
- Commercial use of unauthorized videos

## Recommended Sales Copy

### For Individual Creators

You already own the footage. This tool helps you turn scattered short-video files into a clean local archive with key frames, summaries, tags, and reusable review notes, so each new video improves your next edit.

### For Editors

Use it as a lightweight local review assistant: import authorized client footage, extract visual evidence, generate a structured report, and hand over clearer edit notes without uploading sensitive project files by default.

### For Content Teams / MCNs

Create a private review workflow for authorized team assets. Standardize post-publish review, keep source files local or privately deployed, and build repeatable report templates for creators, editors, and operators.
